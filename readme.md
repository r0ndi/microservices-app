# Microservices application
Project with few basic microservices and one gateway to communicate with client application.
Used technologies: nodejs, be creative and use your favor language for future service.

## To run locally use:
```shell
make pre-run -> prepare and run containers
make sh-ci -> open & go into container
make run -> build and run application
```

- RabbitMQ manager is available on http://localhost:15672/
- Test client application is available on http://localhost:4001/
